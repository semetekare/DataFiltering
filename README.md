# Data filtering
## Описание
Приложение позволяет читать данные из текстовых файлов, фильтровать их по различным критериям и выводить статистику.

Для запуска проекта Java из командной строки нужно иметь скомпилированный байт-код приложения (он находится в target\classes)

- Проект написан в Apache NetBeans 16.
- Используются инструменты JDK 17.
- Система сборки Maven 3.8.6.
## Установка
```shell
git clone https://github.com/semetekare/DataFiltering.git
```
## Компиляция
Для компиляции проекта и последующем создании .class файлов нужно запустить проект из любой IDE

Система сборки должна собрать их за вас при первом запуске

***intelJIDEA:***
- необходимо найти Run/Debug Configurations
- перейти в Edit
- в поле program arguments указать необходимые опции и их значения(при необходимости)
- нажмите apply
- запустите программу

***NetBeans***
- File -> OpenProject... найдите ваш проект
- File -> Project Properties
- Вкладка Run
- поле Arguments

## Запуск
После успешной перекомпиляции(или можно использовать уже собраные файлы, пропустив этап компиляции) вы можете запустить Java-приложение с помощью команды
```shell
cd C:\Users\ "ваш юзер" \ "место расположение репозитория" \target\classes
java com.mycompany.datafiltering.DataFiltering -f -p test_ in.txt
```
## Опции
Программа поддерживает следующие опции командной строки:

- `-o`: Указать путь к каталогу для сохранения результатов.
- `-p`: Указать префикс для имени файлов результатов.
- `-a`: Задать режим добавления данных.
- `-s`: Вывести краткую статистику.
- `-f`: Вывести полную статистику.

## Важно

Файлы для чтения должны находиться в корне запуска программы или иметь абсолютный путь к ним. В противном случае программа может не смочь найти их и выдаст ошибку.
